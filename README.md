# Blog Platform with Comments

## Overview

The Blog Platform is a web application that allows users to create blog posts and interact through comments. The project demonstrates content creation, user interaction, and dynamic web application functionality.

## Features

* Create Blog Posts
* Publish Articles
* Comment on Posts
* Dynamic Content Rendering
* Responsive User Interface
* Interactive User Experience

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

Blog-Platform

├── index.html

├── style.css

├── script.js

└── README.md

## How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open index.html using Live Server.
4. Create blog posts and add comments.
